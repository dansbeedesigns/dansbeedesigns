Parametric Design Meets Solid Cherry Hardwood

This is one of the most challenging products I've designed to date. Not only because the design requires 3D modeling, but also because the design if fully 'parametric'. Parametric implies that a set of parameters drive the final design.

In this case, we have some basic design parameters such as length, width, height, etc., and also a set of parameters driven by a program named Grasshopper that defines the shape of the curves that give Drifting Sounds its beautiful and smooth lines.

Once the design was complete, I could then switch to the next challenge, which was figuring out how to actually make it!

I was able to export the Grasshopper and Rhino 3D files into a format that my [Inventables](https://inventables.com) X-CARVE PRO could then cut out. The maximum cut size for this particular CNC is 48"x48". This initial prototype measures 47.75" x 47.75".
